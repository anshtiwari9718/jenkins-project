# Jenkins CI/CD Project

## Project Description
This project demonstrates a complete CI/CD pipeline using Jenkins, Docker, and GitHub. 
The project automatically clones the repository, builds a Docker image, and runs a container.

## Folder Structure
jenkins-project/
├── Dockerfile
├── Jenkinsfile
└── app/
├── app.py
└── requirements.txt

## Tools & Technologies
- Jenkins
- Docker
- Git & GitHub
- Python (Flask)

## Pipeline Steps
1. Clone the repo from GitHub
2. Build Docker image using Dockerfile
3. Run Docker container
4. Application accessible on port 5000
